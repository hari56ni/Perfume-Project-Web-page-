# 🌸 Perfumy - Perfume Landing Page

## 📌 Project Overview
Perfumy is a simple and responsive perfume landing page developed using **HTML** and **CSS**. It showcases perfume products with a clean user interface, a navigation bar, search section, product cards, an about section, and a contact section.

## 🎯 Features
- Responsive navigation bar
- Search bar
- Product showcase cards
- Hover effect on product cards
- About Us section
- Contact section
- Clean and simple UI design

## 🛠 Technologies Used
- HTML5
- CSS3
